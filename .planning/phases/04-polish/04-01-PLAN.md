# 04-01-PLAN: Command+K Search Implementation

Implement a spotlight search modal that allows users to search through courses, assignments, and app navigation.

## Context
The app needs a quick way to navigate and find information. A spotlight search (Command+K) is a common pattern for productivity apps.

## Tasks

### Task 1: Create Search Component Structure
- [ ] Create `src/components/Search/Search.tsx`.
- [ ] Implement the basic `cmdk` dialog structure.
- [ ] Add the keyboard shortcut listener (Cmd+K / Ctrl+K).
- [ ] Style it with the "Apple Spotlight" look (centered, frosted glass, backdrop blur).

### Task 2: Integrate Data Sources
- [ ] Integrate searching through Navigation items (Dashboard, Schedule, Assignments, Resources).
- [ ] Fetch/Search through Courses from Supabase.
- [ ] Fetch/Search through Assignments from Supabase.
- [ ] Implement multi-tenant filtering (using the current user's session).

### Task 3: Global Integration
- [ ] Add the `Search` component to `src/components/Layout.tsx` or `src/app/AppClient.tsx`.
- [ ] Ensure it works correctly with RTL (Hebrew) and LTR (English).
- [ ] Add icons for different categories (Navigation, Course, Assignment).

## Success Criteria
- [ ] Pressing Cmd+K or Ctrl+K opens the search modal.
- [ ] Users can type to filter results.
- [ ] Results show categorized items (Navigation, Courses, Assignments).
- [ ] Clicking a result navigates to the target page or performs the action.
- [ ] The UI matches the Apple Design System (frosted glass, typography).
- [ ] Escape key closes the modal.
